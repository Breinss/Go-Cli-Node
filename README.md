# Go CLI Application

A barebones CLI application written in Go.

## Installation

```
go install github.com/yourusername/Go-Cli-Node/cmd/app@latest
```

## Usage

```
app [options]
```

## Development

```
git clone https://github.com/yourusername/Go-Cli-Node.git
cd Go-Cli-Node
go build ./cmd/app
```
